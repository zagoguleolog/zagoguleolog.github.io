Раскладка клавиш фортепиано

Схематичная клавиатура: семь белых клавиш на октаву (натуральные ступени), пять чёрных — между ними (паттерн «двойка + тройка»). Подписи — канонические имена и октава в научной нотации. Откройте эту страницу через npm run serve (ES-модули). См. docs/music-theory.md — pitch class и номер ноты MIDI.

Две октавы (C4–B5)

Белые клавиши — натуральные ступени; чёрные — альтерации к соседним белым. Один оборот по белым от C до следующей C — октава, 12 полутонов по клавиатуре.
