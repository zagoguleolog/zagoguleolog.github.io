# Отчёт анализа зигзаг-секвенций

Файл данных: `output/zigzag-sequences.jsonl`  
Строк-секвенций: **396**  

## Проверки реализации

**Ошибок не найдено:** клон lads совпадает со спецификацией по индексам, keys согласованы с indexSpec, скачки индекса только допустимых типов (+1, −1, `2−L`, `L−2`); для всех ладов×тоник базовый ряд из `buildScaleDegreeRowsByMidiOrder` даёт **строго возрастающий MIDI**.

## Эвристика MIDI

Нет соседних пар с |Δ| > 12.
