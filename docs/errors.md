# Реестр ошибок

Фиксируйте только первопричину ошибки, без цепочки вторичных симптомов.

## Формат

```text
ERR-001 | Краткое название
- Дата:
- Контекст:
- Первопричина:
- Решение:
- Статус: open/closed
```

## Правила

- На одну первопричину — одна запись.
- Перед созданием новой записи проверяйте, нет ли уже существующей.
- После фикса обновляйте статус и ссылку на изменение.
