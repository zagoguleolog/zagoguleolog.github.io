# Dev Cheatsheet

## Ежедневный цикл

1. Обновить ветку.
2. Выполнить локальный запуск.
3. Реализовать изменения малыми шагами.
4. Прогнать проверки.

## Минимальные проверки

- Линтер
- Тесты
- Проверка ключевого пользовательского сценария

## Перед ревью

- Убедиться, что документация синхронизирована с кодом.
- Подготовить список изменений и рисков.
