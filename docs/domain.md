# Domain

## Основные сущности

| Сущность | Назначение |
|----------|------------|
| Класс высоты тона (pitch class) | Один из 12 полутонов в октаве, 0…11 от C. |
| Тоника | Опорная нота гаммы; задаёт старт построения по коду шагов. |
| Ступень гаммы | Нота внутри гаммы в пределах октавы относительно тоники. |
| Гамма | Упорядоченный набор ступеней с фиксированными интервалами между соседними ступенями в октаве. |
| Код гаммы | Задание шагов либо как T/t (тоны и полутона), либо как список полутонов; см. [music-theory.md](music-theory.md). Программные сиды и граф: [lib/music-theory.js](../lib/music-theory.js). |
| Интервал | Расстояние между двумя высотами в полутонах; именованные типы в коде — латинские id (`m2`, `M3`, `P5` …); в документации — краткие русские символы и id в таблице (**м2**, **б3**, **ч5** …); см. [music-theory.md](music-theory.md). |
| Триада | Аккорд из трёх звуков, построенный двумя терциями подряд от основания (**б3**+**м3** / `M3`+`m3` — мажор, **м3**+**б3** / `m3`+`M3` — минор); квинта от основания — **ч5** (`P5`); см. [music-theory.md](music-theory.md). |

Подробные правила, таблицы интервалов и каталог гамм — в [music-theory.md](music-theory.md).

## Термины

Единый словарь для кода и документации; расшифровки и примеры — в [music-theory.md](music-theory.md).

| Термин | Определение | Где используется |
|--------|-------------|------------------|
| pitch class (PC) | Номер 0…11 для класса высоты в октаве (C=0). | Построение гамм по алгоритму, таблицы. |
| полутон | Минимальный шаг в 12-тоновой системе. | Код гаммы в числах, интервалы. |
| тон (T) | Целый тон, 2 полутона между соседними ступенями. | Код гаммы T/t. |
| ступень | Элемент гаммы в порядке вверх от тоники. | Каталог гамм, таблицы букв. |
| тоника | Первая ступень гаммы; задаёт высоту старта. | Все построения. |
| унисон (P1), октава (P8) | Интервалы 0 и 12 полутонов; кратко **0** и **8** в паре с таблицей в music-theory.md. | Таблица интервалов. |
| м2, б2 (`m2`, `M2`) | Малая и большая секунда. | Интервалы, пентатоника (б2 / `M2` между ступенями). |
| м3, б3 (`m3`, `M3`) | Малая и большая терция. | Интервалы, пентатоника, триады (**б3**+**м3** / **м3**+**б3**). |
| триада | Аккорд из трёх звуков (две терции подряд от основания). | [music-theory.md](music-theory.md), раздел «Триады». |
| мажорная триада | **б3** + **м3** (`M3` + `m3`) от основания; квинта от основания — **ч5** (`P5`). | Согласованно с таблицей интервалов в music-theory.md. |
| минорная триада | **м3** + **б3** (`m3` + `M3`) от основания; квинта от основания — **ч5** (`P5`). | Согласованно с таблицей интервалов в music-theory.md. |
| относительные тональности | Две тональности с одной знаковой присвоенностью и разной тоникой; в англ. **relative keys**. | [music-theory.md](music-theory.md), раздел «Тональности». |
| параллельные тональности | Две тональности с одной буквой тоники и разным ладом; в англ. **parallel keys**. | [music-theory.md](music-theory.md), раздел «Тональности». |
| диатонические триады | Триады на ступенях лада без хроматических альтераций; корни — те же PC, что у ступеней гаммы тональности. | Круг тональности, [music-theory.md](music-theory.md). |
| спица (доля круга) | Одна доля угла между соседними радиусами; на интерактивном круге — пара подписей (внешнее и внутреннее кольцо) и две группы `.cof-sector` с одним `data-index`. | [music-theory.md](music-theory.md), страницы круга в `app/`. |
| ч4, ч5 (`P4`, `P5`) | Чистая кварта, чистая квинта. | Таблица интервалов, триады (квинта — **ч5**). |
| у5 (`d5`, TT) | Уменьшённая квинта / тритон, 6 полутонов. | Таблица интервалов. |
| м6, б6, м7, б7 (`m6`, `M6`, `m7`, `M7`) | Малые и большие сексты и септимы. | Таблица интервалов. |
| гамма | Лад в пределах октавы с заданным паттерном шагов. | Каталог в music-theory.md. |
| пентатоника | Гамма из пяти ступеней за октаву. | Минорная пентатоника в каталоге. |
| код гаммы | T/t или список полутонов между соседними ступенями (сумма = 12). | Спецификация построения. |
| энгармоника | Разные буквенные записи одного PC (C# / Db). | Соглашения таблиц в music-theory.md. |
| опорная частота A4 | Частота ноты A в четвёртой октаве (научная нотация); задаёт камертон в Гц. | Расчёт частот в равномерной темпации; см. [music-theory.md](music-theory.md). |
| равномерная темпация | Деление октавы на 12 равных по отношению частот шагов (полутонов). | Формула частоты от опорной ноты; см. [music-theory.md](music-theory.md). |
| номер ноты MIDI | Целое 0…127: абсолютная высота в полутонах; **C4 = 60**, **A4 = 69** (соглашение как в [music-theory.md](music-theory.md)). | Связь буквы + октавы с частотой в коде. |
| B-system | Хроматическая раскладка кнопок на баяне / кнопочном аккордеоне: **три** ряда, вдоль ряда шаг **м3**, соседние **полутоны** — по диагонали между рядами. | [bayan-b-system.md](bayan-b-system.md), [`lib/bayan-b-system.js`](../lib/bayan-b-system.js). |
| четырёхрядная хроматическая сетка (баян) | **Четыре** параллельных ряда кнопок; вдоль ряда шаг **б3** (4 полутона); соседние **полутоны** по диагонали; модель **мод 4** по номеру ноты MIDI. Не B-system. | [bayan-b-system.md](bayan-b-system.md) (подраздел «Четырёхрядная хроматическая сетка»), [`lib/bayan-b-system.js`](../lib/bayan-b-system.js). |
| ряд (баян) | В B-system — один из **трёх** параллельных горизонтальных рядов кнопок; в **четырёхрядной хроматической сетке** — один из **четырёх**. Подписи **1-й**…**3-й** или **1-й**…**4-й**; при горизонтальном виде для B-system сверху вниз **3-й → 2-й → 1-й**, для четырёхрядной сетки **4-й → 3-й → 2-й → 1-й**. | [bayan-keyboard.html](../app/bayan-keyboard.html). |
| блок (UI синта) | Визуальная секция панели с общим масштабом (`--synth-scale`); контейнер для сетки ячеек. | [synth-ui.md](synth-ui.md), [`app/synth-kit/`](../app/synth-kit/). |
| ячейка (UI синта) | Минимальная клетка сетки в блоке; размер из `--synth-u` и `--synth-scale`. | [synth-ui.md](synth-ui.md). |
| крутилка | Круговой контрол с вертикальным перетаскиванием (рост значения при движении вверх), чувствительность через `pixelsPerFullRange`. | [`app/synth-kit/knob.mjs`](../app/synth-kit/knob.mjs). |
| фейдер | Вертикальная полоса со ползунком; та же модель дельты по оси Y, что у крутилки. | [`app/synth-kit/fader.mjs`](../app/synth-kit/fader.mjs). |
| матрица (чекбоксов) | Сетка `rows`×`cols` из чекбоксов в стиле kit; состояние — булева матрица или плоский массив. | [`app/synth-kit/checkbox-matrix.mjs`](../app/synth-kit/checkbox-matrix.mjs). |
| режим ввода клавиатуры (синт) | На странице круга: **`mode`** — озвучивание **круга** (удержание / фиксация / полифония секторов); **`keyboardMode`** — игра на **экранной клавиатуре и ПК** (в т.ч. **`holdPoly`** — полифония с удержанием по нажатию/отпусканию). | [synth-structure.md](synth-structure.md), [`app/tone-gen-engine.mjs`](../app/tone-gen-engine.mjs). |

## Бизнес-правила

- Сумма полутоновых шагов в коде гаммы за полный оборот октавы равна **12**.
- **Идентификаторы интервалов в коде** (`m2`, `M3`, `P5` …) остаются латинскими; в документации — **краткие русские** обозначения плюс код в таблице (не требуется дублировать русское в каждом идентификаторе в JS). Термины в комментариях — из колонки «Термин» здесь и из [music-theory.md](music-theory.md).
- Каноническая буквенная запись для именованных гамм — по таблицам в [music-theory.md](music-theory.md), без произвольного смешения энгармоник вне оговорённого соглашения.
