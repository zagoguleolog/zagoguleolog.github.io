# Review Checklist

## Функциональная корректность

- Изменение решает исходную задачу.
- Нет регрессий в ключевых сценариях.

## Качество кода

- Код читаем, контракты ясны.
- Нет дублирования и скрытых побочных эффектов.

## Безопасность

- Входные данные валидируются.
- Нет утечек чувствительной информации.

## Документация и тесты

- Документация обновлена.
- Тесты/ручная проверка приложены к отчёту.
