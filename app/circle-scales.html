Круг и тональность

Подсветка тональности (диатоника) не загрузилась (часто при открытии с диска file://). Запустите из каталога music: npm run serve и откройте страницу по http://… — тогда подключится lib/music-theory.js.

На кольцах — аккорды (внешнее: мажорные центры, внутреннее: относительные миноры). Зелёная подсветка по умолчанию — три спицы кластера IV–I–V опорного мажора (шесть аккордов, по два кольца на спице); опционально галочка «седьмая триада» добавляет vii° / ii° на остальной окружности. У относительных тональностей (например C мажор и A нат. минор) кластер на круге тот же геометрически, см. music-theory.md. Синий/голубой выделенный сектор: в режимах удержание и полифония клик по сектору по-прежнему вкл/выкл выделение; в режиме фиксация на круге всегда не больше одного синего сектора — клик переключает на новый аккорд (повторный клик по тому же сектору не глушит звук). Удержание — звук пока нажата кнопка мыши/пера; фиксация — одна триада у выбранного сектора; полифония — несколько выделенных секторов и триад. Режимы круга — блок Круг слева под SVG; под клавиатурой — голоса (моно/поли) и артикуляция: в полифонии «удержание» — несколько нот с ПК и с экрана одновременно (отпускание глушит только свой голос; см. docs/synth-structure.md про rollover клавиатуры). Стоп глушит круг и клавиатуру до следующего действия. Базовая страница без теории: circle-of-fifths.html.

В режиме «Задать тональность…» клик по внешнему сектору задаёт мажорную тональность по корню подписи; по внутреннему — натуральный минор (для «Am» тоника A). Учитывается «A / B». По умолчанию зелёным подсвечиваются только спицы IV–I–V по внешнему кольцу (относительный мажор в режиме нат. минора). Галочка выше — также седьмая диатоническая триада вне этой тройки. Подробнее — docs/music-theory.md. Для своих строк подписей надёжнее форматы как по умолчанию (C, Am …): разбор корня рассчитан на имя ноты и суффикс m у минора; обозначения вроде Bdim, C7, Am7 не гарантируются.
