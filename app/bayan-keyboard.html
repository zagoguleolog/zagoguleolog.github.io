Клавиатура баяна — B-system

Горизонтальное представление трёх рядов (как «лежа на боку» относительно типичной вертикальной схемы). Теория и формулы — docs/bayan-b-system.md; генерация координат — lib/bayan-b-system.js. Откройте через npm run serve в каталоге music (ES-модули).

Диапазон и шаг вёрстки

Сверху вниз: 3-й, 2-й, 1-й ряд. Вдоль ряда — шаг м3 (3 полутона); хроматика — по диагонали между рядами.
