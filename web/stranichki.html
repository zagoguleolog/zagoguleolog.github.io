Главная — карта сайта

Официальная точка входа: список всех “страничек” (HTML) и основных доков (MD) внутри каталога music/. По адресу корня сервера (http://127.0.0.1:4173/) открывается эта страница. Для корректной загрузки ES-модулей используйте npm run serve в music.
